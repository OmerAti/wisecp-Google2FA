# WISECP Google Authenticator 2FA

WISECP için Google Authenticator destekli iki faktörlü doğrulama modülü.

## Özellikler

- Google Authenticator desteği
- QR kod ile kurulum
- Müşteri panelinden 2FA aç/kapat
- Girişte 6 haneli doğrulama kodu
- Kurtarma kodları

## Kurulum

Modül klasörünü buraya yükleyin:

Modül klasörünü buraya yükleyin:

```text
/coremio/modules/Addons/Google2FA/
```

WISECP admin panelinden eklentiyi aktif edin.
